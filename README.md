# The_Number_Guessing_Game-in-Python

🎮 Number Guessing Game (Python)

A simple command-line Number Guessing Game built using Python.
The computer generates a random number between 1 and 100, and the player has to guess the correct number.

The program gives hints like "Higher Number Please" or "Lower Number Please" until the correct number is guessed.
It also shows a score based on the number of attempts taken by the player.

---

🚀 Features

- Random number generation
- Hint system (Higher / Lower)
- Attempt counter
- Score system based on attempts
- Simple command-line gameplay

---

🧠 How the Game Works

1. The computer randomly selects a number between 1 and 100.
2. The player enters guesses.
3. If the guess is incorrect:
   - The program tells whether the number should be higher or lower.
4. The game continues until the correct number is guessed.
5. After guessing correctly, the program shows the number of attempts and the score.

---

🏆 Score System

Attempts| Score
1 – 4| 🏆 Genius
5 – 7| 👍 Good
8+| 😅 Try Again
